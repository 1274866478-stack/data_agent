# XAI Components

可解释 AI 组件。

## Scope
AI 推理路径和证据链可视化组件。

## Members
| 组件 | 职责 |
|------|------|
| `EvidenceChain.tsx` | 证据链 |
| `SourceCitations.tsx` | 来源引用 |
| `CitationCard.tsx` | 引用卡片 |
| `ReasoningQualityScore.tsx` | 推理质量评分 |
| `ReasoningPath.tsx` | 推理路径 |
| `AnswerExplanation.tsx` | 答案解释 |

## Constraints
- 清晰展示 AI 推理过程
- 支持交互式展开
- 高亮关键证据
